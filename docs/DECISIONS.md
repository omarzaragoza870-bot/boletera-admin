# DECISIONS

## 001 - Nuevo Registro

Reemplazamos el concepto de "Nuevo evento" por "Nuevo registro" porque la plataforma ya no administra solamente funciones o eventos. Ahora administra operación.

## 002 - Tipos operativos

Los primeros tipos definidos son:

- Función
- Activación
- Clase
- Ensayo
- Grabación
- Evento especial
- Traslado
- Mantenimiento

## 003 - Agenda como corazón

La Agenda será el centro de operaciones diario. El calendario es solo una vista, no el producto completo.
